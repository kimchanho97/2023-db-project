## 시작 가이드

![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=Python&logoColor=white)

> Requirements: Python3

1. 프로젝트 클론

```bash
git clone https://github.com/kimchanho97/2023-db-project.git
cd 2023-db-project
```

2. 실행

```bash
pip install psycopg2
python mocking.py
python main.py
```
